Proyecto de Tiple Ten que consta de un análisis integral que incluyó la carga y exploración de datos, una meticulosa limpieza y preparación (gestionando valores nulos en operator_id y normalizando formatos), y la definición explícita de Indicadores Clave de Rendimiento (KPIs) con umbrales claros para la ineficiencia. Utilizando Pandas para la manipulación de datos, Matplotlib y Seaborn para visualizaciones impactantes, y Scipy.stats para pruebas estadísticas rigurosas, logramos identificar patrones clave y operadores problemáticos.
